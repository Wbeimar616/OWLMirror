# **App Name**: OwlMirror

## Core Features:

- Comenzar a Compartir: Iniciar el uso compartido de la pantalla con otros dispositivos en la red.
- Seleccionar Aplicación: Elegir qué aplicación compartir; compartir una ventana de aplicación individual o toda la pantalla.
- Administración de Conexiones: Administrar y mostrar los dispositivos conectados para compartir la pantalla.
- Optimizar los parámetros de compartición: Usar la IA como una herramienta para optimizar los parámetros de compresión en tiempo real de acuerdo con el ancho de banda cambiante y las características del dispositivo para optimizar la transmisión.

## Style Guidelines:

- Color primario: Azul suave (#64B5F6), que promueve una sensación de calma y conectividad.
- Color de fondo: Gris claro (#F0F4F8), que proporciona un telón de fondo neutral.
- Color de acento: Azul eléctrico (#7BC4F5) para elementos interactivos, atrayendo la atención del usuario.
- Fuente del cuerpo y del título: 'Inter' sans-serif para una legibilidad clara y moderna.
- Usar iconos simples y planos para representar acciones y estados.
- Diseño limpio e intuitivo, que prioriza la facilidad de navegación y control.
- Transiciones suaves y animaciones sutiles para mejorar la experiencia del usuario.
